# Prompt Examples

## Feature Work

```text
Use $openspec-superpowers to add team-level usage limits to this OpenSpec repo. Create the OpenSpec change first, then implement with tests and verification.
```

## Bug Fix

```text
This repo uses OpenSpec. Use $openspec-superpowers to fix the export job retry behavior and update the relevant spec if the behavior changes.
```

## Review Follow-Up

```text
Use $openspec-superpowers to address the PR feedback. Keep the OpenSpec artifacts aligned with any behavior changes and verify before reporting completion.
```

## Planning Only

```text
Use $openspec-superpowers to plan this migration. Stop after the OpenSpec proposal, design, specs, and tasks are ready for review.
```

## Degraded Environment

```text
Use $openspec-superpowers. If OpenSpec CLI or delegation tools are unavailable, continue locally and clearly report the deviation.
```
